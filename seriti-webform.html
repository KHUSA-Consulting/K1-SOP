Welcome to the KHUSA Case Submission form. This guide will help you understand how to submit employee-related cases to KHUSA Consulting.

What Is This Form For?

This form enables HR and Payroll personnel to submit cases related to:

  • Healthcare queries
  • Retirement fund queries
  • Employee insurance queries

It ensures that all employee information is accurate and up to date while maintaining a proper record of all submitted cases.

How to Submit a Case?

1. Select the Operation

Choose the relevant operation from the dropdown menu. This helps assign your request to the correct operation.


2. Enter HR/Payroll Contact Details

  • First Name
  • Last Name
  • Email Address

These details allow us to identify who submitted the case and where to send updates.


3. Enter Employee Details

You can search for an employee using either:

  • Payroll Number
  • ID/Passport Number

The system will automatically:

  • Check if the employee exists in our records
  • Auto-populate their details if found
  • Allow entry of new information if not found
Employee not found: All fields are open to be completed by HR.


Employee found without provider data: Auto-populated fields can be edited to update existing information.


Employee found with provider data: Some fields are locked to prevent accidental changes in the KHUSA CRM system.


4. Complete the Case Details

  • Select a Category: Healthcare, Retirement, or Employee Insurance
  • Provide a clear description of the case
  • Attach any relevant documents (optional):
    • Multiple files supported
    • Formats: PDF, Word documents, images
    • Max file size: 10MB per file

After submission, you will receive:

  • A case reference number

What Happens After Submission?

A KHUSA agent will:

  • Review your case.
  • Contact you if additional information is required.
  • Process your request based on the case type and description provided.

You can track your case using the reference number provided.

  • Click the "Follow Up" button on the form to track the progress of your case.
