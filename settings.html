K1 Settings

This settings guide serves as a comprehensive framework for system administrators to effectively manage and optimise the K1 (QContact) system within the organisation.

Address Book

  1. Click the Settings button.
  2. Click the Address Book button.
  3. There is a filter allocated the column to narrow down the search.
  4. Click the columns button to add or remove columns from the view.
  5. Click the “Export to CSV” button to export the data to a CSV file.
  6. Click the live spreadsheet button to export the data in CSV format real time. (clicking this button will copy a link to the clipboard which can be pasted used in the perverted data processing app).
  7. Click the start list button to view the contacts in list form.
  8. Search for an entry using the search box in the top right corner of the view.
  9. Refresh the list by clicking the refresh button in the top right corner of the screen.
  10. Add a new address by clicking the “+” button in the top right corner of the screen.

Step 1: Click the Settings button.

Step 2: Click the Address Book button.

Step 3: Use the filtering options to refine the information displayed based on your criteria.

Columns

  1. Click the columns button to add or remove columns from the view.

By selecting the different columns, the layout of the view can be adjusted accordingly.

Export to csv

  1. Click the “Export to CSV” button to export the data to a CSV file.

By utilizing the "Export to CSV" button, you can download data to a CSV file, which will be saved in the "Downloads" folder on the computer.

Live spreadsheet

  1. Click the live spreadsheet button to export the data in CSV format real time. (clicking this button will copy a link to the clipboard which can be pasted used in the prevented data processing app).
  2. Select with time zone or without time zone (With Time zone is commonly used).

Step 1: Click the live spreadsheet button

Step 2: Select a Time zone

Start list

This feature enables the user to view the entire address list in a card format. Users can iterate through the list by following the arrow to navigate to the next address card.

  1. Click on start list.
  2. View the addresses one at a time.

Step 1: On the start list you can view the different addresses.

Step 2: View the individual addresses one at a time.

Global search

The address book page includes a dedicated search function specifically for the address entity. This allows users to efficiently locate specific addresses by entering relevant search criteria.

Refresh

The Refresh Button allows the user to update the data table, ensuring that the latest entries are included in the view. Click this button to retrieve the most current information available.
