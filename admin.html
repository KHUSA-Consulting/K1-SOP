Change ringer

This will enable the functionality to change the ringing destination. If there is a headset plugged into the computer, it is easy to miss a call due to the device ringing on the headset. Changing the ringer will allow a call to ring on the computer’s physical speaker even with a headset plugged in.

  1. Click the change ringer button.
  2. Choose the appropriate option for selecting the ringer output:
    1. If "Default" is selected, the system will use the default audio device.
    2. If "Communications Default" is selected, this will be used for audio related to communication apps (e.g., calls, meetings).
    3. If "Speakers" is selected, audio will be played through connected speakers or headphones using the Realtek audio driver.
    4. If “Headphones” are selected, the device will ring through the connected headphones.

Step 1: Click the change ringer button.

Step 2: Choose between default, communications default, speakers or Headphones.

Queue

The virtual line where incoming interactions or customer requests are placed (calls, emails, messages) while waiting to be handled by available agents.

  1. Click the "Go on Queue" Button: Select the "Go on Queue" button to become available for incoming requests and join the active interaction queue.
  2. Click the "Go off Queue" Button: When you need to stop receiving requests, click the "Go Off Queue" button to exit the call queue and become unavailable for further calls.

Click the "Go on Queue" Button: This button will turn into a green “Play” button to indicate that the user is on queue.

Click the "Go off Queue" Button: This button will turn into a red “Pause” button to indicate that the user is off queue.

Voice call handling

  1. Answer or Make the Call: Accept an incoming call or initiate a call to the desired contact.
  2. Tag the Call: Designate the call as either "Member" or "Corporate" for categorisation purposes (this function is not commonly used).
    1. Click the “Tag” button.
    2. Select member or corporate.
    3. Click the big purple Tag button.
  3. Hold: Put the call on hold if you need to pause the conversation temporarily.
    1. Click the “Hold” button to place the call on “Hold”.
    2. Click the “Unhold” button to continue the call.
  4. Transfer: Transfer the call to another user if the call needs to be directed elsewhere.
    1. Click the “Transfer” button.
    2. Click the one of the following”:
      1. User.
      2. Department (Not commonly used).
      3. External (Not commonly used).
      4. Self Service (Not commonly used).
    3. Search for the required user by:
      1. Typing the username in the text box.
      2. Scrolling through the user list
    4. Select the internal transfer option.
    5. Click the required transfer button:
      1. Blind Transfer: This will end the call on the current user’s side and ring with the new user.
      2. Warm Transfer: This will keep the current user on the line until the new user answers the call.
  5. Add: Add another contact to the call for a conference conversation.
    1. Click the “Add” button.
    2. Complete the contact number in the text box.
    3. Click the call Button.
  6. Re-assign: Assign the call to the correct contact if it was incorrectly assigned.
    1. Click the “Re-Assign” button.
    2. Search for the contact by typing the contact details in the text box. The following can be used to search for the contact card.
      1. I. ID number.
      2. II. First Name.
      3. III.Last Name.
      4. IV. Contact Number.
      5. V. Email Address.
      6. VI. Membership number.
      7. VII.Payroll number.
    3. Click the big purple “Re-Assign” Button.
  7. Callback Request: Create a callback request if the contact has requested a follow-up call.
    1. Click the “Callback” button.
    2. Complete the “Callback” details (Refer to the callback creation section).
  8. Tag Conversation: Tag the conversation to an existing case or create a new case if necessary for documentation (Refer to the case creation section).
    1. Click the edit (Pencil) button on the “Tag Conversation” bar.
    2. Search for the Case by:
      1. Typing the case details in the text box.
      2. Scrolling through the case list
    3. Click the green tick mark next to the text box to finalise the selection.
  9. Disposition: Dispose of the call when finished. This action will indicate the reason for the call and will also end the call.
    1. Click the Disposition button.
    2. Select the required Disposition.
    3. Click the big purple disposition button.
  10. Hang Up: Hang up the call when completed. This action will prompt the disposition menu, allowing you to select the required reason for the call within a specified time limit (15 seconds).
    1. Click the Hangup button.
    2. Select the required Disposition.
    3. Click the big purple disposition button.
